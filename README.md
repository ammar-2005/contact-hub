# 📒 Contact Hub — Smart Contact Manager

A clean and responsive contact manager web app built with **HTML**, **CSS**, **Bootstrap 5**, and **Vanilla JavaScript**. All data is saved locally in the browser using `localStorage` — no backend needed.

---

## ✨ Features

- ➕ Add new contacts (name, phone, email, address, group, notes)
- ✏️ Edit existing contacts
- 🗑️ Delete contacts
- ⭐ Mark contacts as Favorites
- 🚨 Mark contacts as Emergency
- 🔍 Search contacts by name, phone, or email
- 📊 Live stats (Total / Favorites / Emergency)
- 📋 Sidebar with quick access to Favorites & Emergency 



## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling |
| Bootstrap 5 | Layout & UI components |
| Font Awesome | Icons |
| Vanilla JavaScript | Logic & interactivity |
| SweetAlert2 | Beautiful alerts |
| localStorage | Data persistence |

---

## 📸 How to Use

1. Click **+ Add Contact** in the navbar
2. Fill in the contact details and click **Save Contact**
3. Use the ⭐ button on a card to add to **Favorites**
4. Use the 💗 button on a card to add to **Emergency**
5. Use the ✏️ button to **edit** a contact
6. Use the 🗑️ button to **delete** a contact
7. Use the **search bar** to filter contacts instantly
